## Calculator (JavaScript)

A basic calculator application built using HTML, CSS, and JavaScript. It performs standard arithmetic operations such as addition, subtraction, multiplication, and division.

### Features
- Handles basic arithmetic operations
- Interactive UI with button inputs
- Real-time display updates
- Error handling for invalid operations (e.g., division by zero)

### Tech Stack
- HTML
- CSS
- JavaScript
